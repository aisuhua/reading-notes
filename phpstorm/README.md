# Phpstorm

## 快捷键

- ctrl + r 查看替换当前文件
- ctrl + shift + r 查看替换整个项目
- ctrl + shift + u 大小写切换
- ctrl + / 行注释
- ctrl + shift + / 块注释
- ctrl + d 复制当前行
- ctrl + y 删除当前行

## 技巧

- 切换项目所支持的 PHP 版本

  Settings -> Languages & Frameworks -> PHP -> Development Environment -> PHP language level

- 切换主题

  Settings -> Editor -> Colors & Fonts -> Scheme -> Solarized Light -> Font (Source code Pro 16px  Line spacing 1.0)
